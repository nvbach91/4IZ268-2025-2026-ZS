O Projekte a Autorovi

Kontext a Pôvod Hry

Táto statická webová prezentácia vznikla ako Semestrálny Projekt (SP1) pre predmet 4IZ268: Webové technológie na Vysokej škole ekonomickej v Prahe.

Téma webstránky čerpá z projektu Textová Adventúra ("Tajomstvo Zatopeného Chrámu"), ktorú autor vytvoril pre predmet 4IT115: Softvérové inžinierstvo.

Informácie o Autorovi

  • Autor: Andrej Braun
  • Študijný program: Aplikovaná informatika
  • Univerzita: VŠE Praha (Fakulta informatiky a statistiky)
  • Predmet SP1: 4IZ268 – Webové technológie

Odkazy a Kód

Web je plne responzívny, využíva CSS Grid/Flexbox pre dvojslĺpcový layout (detaily miestností) a modulárne CSS štýly. Spĺňa sémantiku HTML5 a neobsahuje externé CSS/JS frameworky.

  • Kód Hry (4IT115): Zdrojový kód Textovej Adventúry
