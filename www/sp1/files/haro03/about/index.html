O webu

Proč tento web vznikl

Tento web byl vytvořen jako jednoduchý průvodce pro začátečníky, kteří se chtějí naučit správně pečovat o pokojové rostliny. Cílem je poskytnout přehledné a srozumitelné informace o základní péči, nejčastějších typech rostlin a jejich specifických potřebách.

Co zde najdete

Web je rozdělen do několika částí, které se věnují různým aspektům pěstování pokojových rostlin:

  • Domovská stránka: stručný úvod a navigace;
  • Péče: informace o světle, zalévání, hnojení a přesazování;
  • Časté problémy: rešení pro bežné problémy;
  • Druhy rostlin: základní přehled nejčastějších skupin pokojových rostlin;
