Akademie podnikání

Klient

Česká spořitelna a.s.

Rok

2025 - současnost

Služby

UI/UX Design, analýza

Nástroje

Figma, HTML, CSS

O projektu

Pro Českou spořitelnu jsem navrhoval vizuální podobu jejich veřejné Akademie podnikání. Jedná se o vzdělávací portál pro veřejnost, který je technologicky postavený na našem LMS systému Scormium.

Výzva

Zadáním bylo vytvořit atraktivní a důvěryhodný frontend, který bude plně respektovat vizuální identitu banky. Museli jsme ohnout standardní šablony LMS systému tak, aby výsledek nepůsobil jako korporátní systém, ale jako moderní, vzdušný web pro podnikatele.

Moje role

Navrhl jsem kompletní UI design veřejné části portálu. Soustředil jsem se na prezentaci katalogu kurzů a detailů školení. Cílem bylo, aby uživatel na první pohled nepoznal, že se pohybuje v LMS systému, ale vnímal web jako integrální součást ekosystému České spořitelny.
