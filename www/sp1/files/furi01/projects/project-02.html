COVID-19 Štatistický Dashboard

Jednoduchý dashboard, ktorý sleduje základné čísla o pandémii a skúša ukázať trendy bez zbytočne komplikovaných grafov.

Čo som chcel zistiť

Skôr než „vedecký výskum“ išlo o skúšanie, ako sa menia denné prírastky, sedemdňové priemery a ako to celé vyzerá na mape.

Nástroje a dáta

SQL Power BI REST API ETL

Jednoduchý dotaz

SELECT
  report_date,
  country,
  new_cases,
  new_deaths
FROM covid_daily
WHERE country = 'Slovakia'
ORDER BY report_date;

Dashboard ukazuje pár základných kariet, jednoduchý čiarový graf a tabuľku. Cieľom bolo „vidieť čísla“, nie postaviť ideálny BI systém.
