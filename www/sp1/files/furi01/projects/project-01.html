Analýza spotrebiteľského správania v e-commerce

Detailný pohľad na nákupné správanie zákazníkov, prediktívne modely a praktický dashboard pre marketingový tím.

Ciele projektu

Cieľom bolo identifikovať kľúčové segmenty zákazníkov, pochopiť ich nákupné správanie a pripraviť odporúčania pre retenčné kampane.

Použité nástroje

Python Pandas SQL Tableau

Ukážka transformačnej logiky

WITH orders_enriched AS (
  SELECT
    customer_id,
    order_id,
    order_date,
    revenue
  FROM orders
)
SELECT
  customer_id,
  SUM(revenue) AS clv
FROM orders_enriched
GROUP BY customer_id;

Výstupom bol jednoduchý dashboard, ktorý umožňuje stakeholderom filtrovať zákazníkov podľa hodnoty CLV a sledovať vývoj churnu v čase.
