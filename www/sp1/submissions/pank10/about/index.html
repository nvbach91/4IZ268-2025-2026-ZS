O archivu

Folklórní archiv vílího světa vznikl jako semestrální projekt pro předmět 4IZ268 - Webové technologie na Vysoké škole ekonomické v Praze. Cílem projektu je demonstrovat možnosti moderních webových technologií při vytváření vzdělávacího obsahu s důrazem na estetiku a přístupnost.

Archiv se zaměřuje na studium víl a vílích bytostí v evropském folklóru, mytologii a umění. Projekt spojuje akademický přístup k tématu s moderním webdesignem inspirovaným estetikou Dark Academia a Cottagecore.

Technické informace

Použité Technologie

  • HTML5 – Sémantické značkování pro lepší přístupnost a SEO
  • CSS3 – Moderní layout s využitím CSS Grid a Flexbox
  • Responzivní design – Optimalizace pro různé velikosti obrazovek
  • Print CSS – Speciální styly pro tisk dokumentů

Standardy a validace

Web je vytvořen v souladu s webovými standardy W3C a je validní podle:

  • CSS3 Validator - Blahopřejeme! Chyby nenalezeny!
  • WCAG 2.1 pro přístupnost - Accessibility Score 94 - úroveň AA
