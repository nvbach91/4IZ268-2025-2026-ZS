Welcome to Movie Library

Movie Library is a simple web application where you can search movies, read basic information and save your favourites.

Start searching movies

About the project

This website was created as a semester project for the Web Technologies course at the University of Economics in Prague. The main goal was to build a small but functional movie library using only HTML, CSS, JavaScript and the Bootstrap framework.

The application communicates with TMDB (The Movie Database) API, which provides real movie data such as titles, posters, ratings and release dates.

What you can do here

  • Search for movies by title and filter results by year, rating and sorting.
  • Open a detail page with more information about a selected movie.
  • Save movies to your favourites and keep them even after reloading the page.
  • Quickly navigate between Home, Search, Details, Favourites and Team pages.

Used technologies

  • HTML5 and CSS3 for structure and styling.
  • Bootstrap 5 for responsive layout and basic components.
  • Vanilla JavaScript for working with TMDB API and browser localStorage.
  • TMDB API for movie data (titles, posters, ratings, overview).
